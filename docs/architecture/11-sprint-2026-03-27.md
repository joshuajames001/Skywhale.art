1. **Dual-variant pattern**: Orchestrátor + Desktop + Mobile
2. **SharedProps interface**: Jeden props object pro obě varianty

4. **Reusable BottomSheet**: Komponenta v CustomBookEditorMobile
5. **Purple accent system**: #534AB7 / #EEEDFE / #AFA9EC
6. **Panel extraction pattern**: Monolitní mobile → orchestrátor + panels/ + sheets/
